# UI 紧凑化更新

## 更新内容

优化了文书智能生成对话框的布局，减少了不必要的空白区域，使界面更加紧凑。

## 修改的文件

- `frontend/src/components/document/SmartDocumentGenerator.vue`

## 具体调整

### 1. 整体容器
- 上下内边距：20px → 10px
- 步骤内容上边距：30px → 15px
- 最小高度：400px → 300px

### 2. 模板选择区域
- 标题下边距：20px → 12px
- 标题字体大小：默认 → 16px
- 卡片内边距：20px → 12px 8px
- 卡片下边距：20px → 12px
- 图标大小：40px → 32px
- 图标下边距：15px → 8px
- 名称下边距：8px → 5px
- 描述字体大小：13px → 12px
- 描述行高：默认 → 1.4

### 3. 信息确认区域
- 提示框下边距：20px → 12px
- 表格大小：默认 → small
- 表单标签宽度：120px → 100px
- 表单大小：默认 → small
- 表单上边距：20px → 12px
- 诉讼主体标题：添加紧凑样式

### 4. 文书生成区域
- 成功提示下边距：20px → 12px

## 效果

- 对话框整体高度减少约 20-25%
- 保持了良好的可读性和可用性
- 视觉层次更加清晰
- 减少了滚动需求
