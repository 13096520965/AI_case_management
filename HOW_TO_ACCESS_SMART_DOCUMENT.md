# 如何访问智能文书生成与审核功能

## 🎯 功能入口

智能文书生成与审核功能已经集成到**案件详情页**中。

## 📍 访问步骤

### 方法1：从案件列表进入

1. 访问系统首页：http://localhost:5173
2. 登录系统（用户名：admin，密码：admin123）
3. 点击左侧菜单的"案件管理" → "案件列表"
4. 在案件列表中点击任意案件的"查看"按钮
5. 在案件详情页中，找到**"案件文书管理"**模块（带有绿色"智能AI辅助"标签）

### 方法2：直接访问案件详情

访问：http://localhost:5173/cases/1

（将数字1替换为任意案件ID）

## 🚀 功能使用

### 智能生成文书

1. 点击**"智能生成"**按钮
2. 选择文书类型（起诉状、答辩状、代理词等）
3. 确认案件信息和诉讼主体
4. 填写代理律师信息（可选）
5. 点击"下一步"，AI自动生成文书
6. 可以在线编辑、复制或下载
7. 点击"保存文书"保存到系统

### 智能审核文书

1. 点击**"智能审核"**按钮
2. 选择"上传文件"或"直接输入"
3. 上传文书文件或粘贴文书内容
4. 选择审核选项（合规性、逻辑性、格式、完整性）
5. 点击"开始智能审核"
6. 查看审核结果和问题列表
7. 可以导出审核报告

### 文书管理

在文书列表中，你可以：
- 查看已保存的文书
- 下载文书
- 删除文书

## 💡 提示

- **绿色标签**：表示这是AI辅助功能
- **智能生成**：基于案件信息自动生成规范文书

## 🔧 技术说明

- 前端：Vue 3 + TypeScript + Element Plus
- 后端：Node.js + Express
- 数据库：SQLite（smart_documents表）
- AI：当前使用模拟AI，可接入真实AI服务
