## 提醒系统功能

### 文案格式
```
【超期警告】冀1091民初8255号案件节点"第一次庭审"已超期 662 天（截止日期：2024-02-01 17:00:00），请尽快处理。
（下方字段）关联案件编码:AN202511000002
```

### 功能特性
✓ 文案清晰，不包含关联案件编码
✓ 关联案件编码显示在下方字段
✓ 没有内部编号时显示 `--`
✓ 点击关联信息可跳转到案件详情页面
✓ 右上角通知宽度不足时自动折行显示
