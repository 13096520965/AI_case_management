## 交互流程

### 1. 选择实体类型
```
点击 [企业] → 显示企业相关字段（统一社会信用代码、法定代表人）
点击 [个人] → 显示个人相关字段（身份证号）
```

### 2. 选择地区
```
点击地区选择器 → 展开省级列表
选择省份 → 展开市级列表
选择城市 → 展开区级列表
选择区县 → 完成选择，自动关闭
```

### 3. 提交表单
```
点击 [确定] → 验证所有必填项
验证通过 → 提交数据 → 显示成功提示 → 关闭弹窗 → 刷新列表
验证失败 → 显示错误提示 → 保持弹窗打开
```

## 响应式设计

### 桌面端 (>1200px)
- 表单宽度：700px
- 标签宽度：120px
- 输入框占满剩余空间

### 平板端 (768px-1200px)
- 表单宽度：90%
- 标签宽度：100px
- 输入框自适应

### 移动端 (<768px)
- 表单宽度：95%
- 标签上下布局
- 输入框全宽

## 颜色方案

- 主色调：Element Plus 默认蓝色 (#409EFF)
- 成功：绿色 (#67C23A)
- 警告：橙色 (#E6A23C)
- 错误：红色 (#F56C6C)
- 信息：灰色 (#909399)

## 图标使用

- ✓ 验证成功
- ⚠ 验证警告
- ✕ 关闭/删除
- ▼ 下拉展开
- ▶ 级联展开
- + 添加
