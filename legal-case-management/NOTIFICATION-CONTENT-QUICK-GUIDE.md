# 提醒文案优化 - 快速指南

## 改进内容

### 原文案
```
【超期警告】节点"送达判决书"已超期 6 天（截止日期：2025-11-19T02:25:32.365Z），请尽快处理
关联: process_node #23
```

### 新文案
```
【超期警告】AN202511000001案件节点"送达判决书"已超期 6 天（截止日期：2025-11-19 02:25:32），请尽快处理
关联: 案件内部编号 AN202511000001
```

## 改进点

| 项目 | 原 | 新 | 说明 |
|------|-----|-----|------|
| 案号 | ❌ 无 | ✓ AN202511000001 | 显示案件内部编号 |
| 日期格式 | ISO格式 | YYYY-MM-DD HH:mm:ss | 更易读 |
| 关联信息 | process_node #23 | 案件内部编号 | 更清晰 |
| 跳转功能 | ❌ 无 | ✓ 点击跳转到案件详情 | 提高用户体验 |
