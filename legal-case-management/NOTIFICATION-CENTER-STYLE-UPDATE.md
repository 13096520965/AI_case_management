# 提醒中心样式更新

## 修改内容

将 `/notifications` 页面（NotificationCenter）的样式更新为与首页待办事项一致的简洁风格。

## 主要变更

### 1. 移除背景色区分

**修改前：**
```css
.notification-item.unread {
  background-color: #f0f9ff;  /* 未读有蓝色背景 */
}

.notification-item.unread:hover {
  background-color: #e6f4ff;
}
```

**修改后：**
```css
/* 移除未读背景色样式 */
/* 所有消息使用统一的白色背景 */
```

### 2. 移除紧急边框

**修改前：**
```css
.notification-item.urgent {
  border-left: 3px solid #f56c6c;  /* 紧急消息有红色边框 */
}
```

**修改后：**
```css
/* 移除 border-left 样式 */
```

### 3. 添加未读红点

**模板修改：**
```html
<!-- 修改前 -->
<div
  class="notification-item"
  :class="{
    'unread': notification.status === 'unread',
    'urgent': isUrgent(notification)
  }"
>
  <div class="notification-icon">
    <el-icon :size="24" ...>
  </div>
</div>

<!-- 修改后 -->
<div class="notification-item">
  <!-- 添加红点 -->
  <div class="item-dot" v-if="notification.status === 'unread'"></div>
  
  <div class="notification-icon">
    <el-icon :size="20" ...>  <!-- 图标尺寸调整为20 -->
  </div>
</div>
```

**样式添加：**
```css
.notification-item {
  position: relative;  /* 添加相对定位 */
  /* ... 其他样式 */
}

.item-dot {
  position: absolute;
  left: 8px;
  top: 22px;
  width: 6px;
  height: 6px;
  border-radius: 50%;
  background-color: #f56c6c;
}
```

## 优势

1. **视觉统一** - 与首页待办事项样式完全一致
2. **界面简洁** - 移除多余的背景色和边框
3. **重点突出** - 红点清晰标识未读状态
4. **用户体验** - 统一的交互模式，降低学习成本

## 测试要点

1. ✅ 未读消息显示红点
2. ✅ 已读消息无红点
3. ✅ 没有背景色区分
4. ✅ 没有 border-left 样式
5. ✅ 图标颜色根据类型变化

7. ✅ 点击"标记已读"后红点消失

## 注意事项

1. 红点位置为 `left: 8px, top: 22px`（比Dashboard稍低，因为padding不同）
2. 图标大小调整为 20px（与Dashboard的18px接近）
3. 移除了 `unread` 和 `urgent` 类名的样式
4. 保持了原有的功能（筛选、分页、标记已读、删除等）
