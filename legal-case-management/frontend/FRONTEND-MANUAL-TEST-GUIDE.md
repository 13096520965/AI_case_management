# 前端手动测试指南

本指南提供了快速验证案例知识库前端功能的步骤。

## 前提条件

1. 后端服务运行在 `http://localhost:3000`
2. 前端服务运行在 `http://localhost:5173`
3. 已有测试账号可以登录

## 快速测试步骤

### 第一步：验证导航菜单

1. 登录系统
2. 查看左侧导航菜单
3. **验证点**：
   - [ ] 能看到"案例知识库"一级菜单项
   - [ ] "案例知识库"位于"归档管理"之前
   - [ ] 菜单项有书籍或笔记本图标
   - [ ] "归档管理"菜单中不再有"案例知识库"子菜单

### 第二步：验证路由跳转

1. 点击"案例知识库"菜单项
2. **验证点**：
   - [ ] 页面正常跳转
   - [ ] URL 显示为 `/knowledge`
   - [ ] 页面显示知识库列表

3. 在浏览器地址栏输入 `http://localhost:5173/archive/knowledge`
4. **验证点**：
   - [ ] 自动重定向到 `/knowledge`
   - [ ] 页面正常显示

### 第三步：验证CRUD功能

#### 创建知识条目
1. 点击"新增知识"按钮
2. 填写表单：
   - 案由：劳动争议
   - 争议焦点：加班费计算
   - 其他字段可选填
3. 点击"确定"
4. **验证点**：
   - [ ] 显示"创建成功"提示
   - [ ] 列表自动刷新
   - [ ] 新记录出现在列表中

#### 查看知识详情
1. 点击任意记录的"查看"按钮
2. **验证点**：
   - [ ] 弹出详情对话框
   - [ ] 显示完整的知识信息
   - [ ] 所有字段正确显示

#### 编辑知识条目
1. 点击任意记录的"编辑"按钮
2. 修改某些字段
3. 点击"确定"
4. **验证点**：
   - [ ] 显示"更新成功"提示
   - [ ] 列表自动刷新
   - [ ] 修改后的内容正确显示

#### 删除知识条目
1. 点击任意记录的"删除"按钮
2. 确认删除
3. **验证点**：
   - [ ] 显示确认对话框
   - [ ] 显示"删除成功"提示
   - [ ] 记录从列表中消失

### 第四步：验证搜索功能

1. 在搜索框输入关键词（如"劳动"）
2. 点击"搜索"按钮
3. **验证点**：
   - [ ] 列表显示匹配的结果
   - [ ] 结果包含搜索关键词
   - [ ] 清空搜索后显示所有记录

### 第五步：验证分页功能

1. 如果记录超过10条，查看分页器
2. 点击"下一页"
3. **验证点**：
   - [ ] 分页器正确显示
   - [ ] 翻页功能正常
   - [ ] 页码正确更新

### 第六步：检查浏览器控制台

1. 打开浏览器开发者工具（F12）
2. 切换到 Console 标签
3. 执行上述所有操作
4. **验证点**：
   - [ ] 无红色错误信息
   - [ ] 无黄色警告信息（或仅有可忽略的警告）

5. 切换到 Network 标签
6. 执行一些操作（如创建、查询）
7. **验证点**：
   - [ ] 请求路径为 `/api/knowledge`
   - [ ] 所有请求返回 200 或 201 状态码
   - [ ] 响应数据格式正确
