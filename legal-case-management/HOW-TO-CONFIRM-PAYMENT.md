# 如何确认汇款到账 - 快速指南

## 问题

汇款记录显示"已汇款"状态，金额1000元，但"已收回"显示为0元。

## 原因

**这是正常的！** 只有状态为"**已确认**"的汇款才会计入"已收回"金额。

## 解决方案

将汇款记录的状态从"已汇款"改为"已确认"。

## 操作步骤

### 方法1：在详情对话框中修改

1. **打开标的处理详情**
   - 进入案件编辑页面
   - 找到"标的处理详情"卡片
   - 点击"查看详情"按钮

2. **切换到汇款记录标签页**
   - 在弹出的对话框中
   - 点击"汇款记录"标签

3. **编辑汇款记录**
   - 找到金额为1000元的记录
   - 点击该记录行的"编辑"按钮

4. **修改状态**
   - 在弹出的编辑表单中
   - 找到"状态"下拉框
   - 将状态从"已汇款"改为"已确认"

5. **保存**
   - 点击"保存"按钮
   - 系统显示"汇款记录更新成功"

6. **验证结果**
   - 关闭编辑对话框
   - 查看汇款记录列表
   - 该记录的状态标签应该变为绿色的"已确认"
   - 返回卡片查看统计
   - "已收回"应该显示为 ¥1,000.00

## 状态说明

### 🟡 待汇款
- **含义**：计划中的汇款
- **是否计入已收回**：❌ 否

### 🔵 已汇款
- **含义**：对方已汇款，等待确认
- **是否计入已收回**：❌ 否

### ✅ 已确认
- **含义**：已确认到账
- **是否计入已收回**：✅ 是

## 批量确认多笔汇款

如果有多笔"已汇款"状态的记录需要确认：

1. 逐条点击"编辑"
2. 修改状态为"已确认"
3. 保存
4. 重复以上步骤

**提示**：未来版本可能会添加批量操作功能。

## 常见场景

### 场景1：对方刚汇款

```
操作：创建记录 → 状态选择"已汇款"
原因：等待银行到账确认
```

### 场景2：确认到账

```
操作：编辑记录 → 状态改为"已确认"
原因：银行账户已收到款项
```

### 场景3：计划未来汇款

```
操作：创建记录 → 状态选择"待汇款"
原因：对方承诺的未来付款
```

## 为什么这样设计？

1. **财务准确性**
   - 只有确认到账的款项才是真实回收
   - 避免虚报回收率

2. **风险控制**
   - "已汇款"可能存在风险（凭证造假、转账失败等）
   - 确认到账后才计入统计

3. **业务流程**
   - 符合实际财务管理流程
   - 待汇款 → 已汇款 → 已确认

## 快速检查清单

确认汇款到账时，请检查：

- [ ] 银行账户已收到款项
- [ ] 金额与记录一致
- [ ] 付款方信息正确
- [ ] 汇款日期合理
- [ ] 修改状态为"已确认"
- [ ] 保存成功
- [ ] 统计数据已更新

## 总结

**"已汇款"不计入"已收回"是正确的设计！**

要让金额计入"已收回"，请：
1. 确认银行账户已到账
2. 编辑汇款记录
3. 将状态改为"已确认"
4. 保存

系统会自动更新统计数据。✅
